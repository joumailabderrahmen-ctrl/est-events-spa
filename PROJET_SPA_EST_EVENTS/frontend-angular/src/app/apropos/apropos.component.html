Le projet

EST Events est une Single-Page Application full-stack développée dans le cadre du module Développement Web Avancé. Elle centralise la gestion des événements de l'EST Dakhla : découverte, réservation, suivi et administration.

Objectifs atteints
  • Architecture MVC backend + SPA Angular
  • Sécurité JWT, Helmet, CORS, Rate-limiting
  • APIs HTML5 : WebSocket, WebRTC, Geolocation, Canvas
  • 69 tests unitaires Jest + Jasmine (100% passing)
